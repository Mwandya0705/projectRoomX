# 🌌 RoomX: The Monetization Era Sanctuary

**RoomX** is a high-fidelity, hyper-cinematic platform designed for creators, experts, and high-value consultants to build, manage, and monetize private digital sanctuaries. It transforms the standard video conferencing experience into a professional production ecosystem powered by real-time AI intelligence.

---

## 🎭 The Core Experience

### 💎 Hyper-Cinematic Interface
Every interaction in RoomX is a cinematic event.
- **Sanctuary Sync Transitions**: A signature geometric mask and "Deep Teal" curtain reveal that orchestrates every page swap with GSAP-powered precision.
- **Neutral-State Stability**: Custom motion locks that prevent glitches during View Transitions, providing a world-class, fluid user experience.

### 🧠 Sanctuary Intelligence Engine
Turn your live sessions into searchable, high-value assets.
- **Quantum Recording**: One-click audio capture directly from the Studio stage.
- **Surgical Transcription**: Powered by **OpenAI Whisper**, converting every spoken word into a surgical-grade raw transcript.
- **AI Executive Synthesis**: **GPT-4** synthesis that extracts key highlights, action items, and executive summaries instantly.
- **Intel Export**: Download professional Meeting Intelligence Reports (.txt) for your subscribers or archive.

### 💰 The Monetization Hub
Direct revenue generation for creators.
- **Private Sanctuaries**: Build rooms with strict access control and monthly subscription passes.
- **Aesthetic Subscription Gateways**: Cinematic checkout portals that invite users to join the "Monetization Era."
- **Flexible Pricing**: Set monthly access fees (TZS) and manage visibility (Public Discovery vs. Private Sanctuary) with a haptic configuration engine.

### 🎮 Creator Control Center
- **Dual-Room Pagination**: A clutter-free dashboard displaying exactly two high-fidelity rooms per page.
- **Interactive Configuration**: Modify room identity, monetization, and AI features (AI Summaries, 4K Mode) via a glassmorphic modal—no page reloads required.

## 🛠️ Technical Architecture

RoomX is built on a world-class modern stack:

- **Frontend**: [Next.js 14/15](https://nextjs.org/) (App Router), [React 18](https://reactjs.org/)
- **Motion & UI**: [GSAP](https://greensock.com/gsap/) (GreenSock), [Framer Motion](https://www.framer.com/motion/), [Next View Transitions](https://github.com/shuding/next-view-transitions)
- **Live Video/Audio**: [LiveKit](https://livekit.io/) (Real-time WebRTC)
- **Backend & Auth**: [Supabase](https://supabase.com/) (Auth, PostgreSQL, Storage)
- **AI Brain**: [OpenAI](https://openai.com/) (Whisper-1 for Transcription, GPT-4 for Summarization)
- **Payments**: [ClickPesa](https://clickpesa.com/) / [Stripe](https://stripe.com/) Integration

## 🚀 Quick Start

### 1. Environment Configuration
Create a `.env.local` file with the following credentials:

```bash
# Database & Storage
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
SUPABASE_SERVICE_ROLE_KEY=your_secret_key

# Live Studio (LiveKit)
NEXT_PUBLIC_LIVEKIT_URL=your_livekit_url
LIVEKIT_API_KEY=your_api_key
LIVEKIT_API_SECRET=your_api_secret

# AI Intelligence (OpenAI)
OPENAI_API_KEY=your_openai_key

# Monetization
PAYMENT_API_KEY=your_payment_key
```

### 2. Installation
```bash
npm install
npm run dev
```
